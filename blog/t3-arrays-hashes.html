Hashes vs Arrays

Arrays | Creating New Array

Arrays are ordered collections of objects. A common use for arrays is to group similar objects in a list and give them an ordered number starting with 0. You can create a new array by typing in the following code into your ruby text editor:
fruits = ["apple", "orange", "pear"]

Arrays | Adding objects to array

In our example above we created a new array called fruits. We can add more fruits to our array with the following code, here are several different ways:
fruit.push("lemon", "lime")
    => ["apple", "orange", "pear", "lemon", "lime"]

fruit.unshift("lemon", "lime")
    => ["lemon","lime","apple", "orange", "pear"]

Arrays | selecting objects in array

Every value in an array is assigned an index number. in our array fruits = ["apple", "orange", "pear"] the value "apple" has a index value of 0. We can retrieve and print to the screen "apple" with the following code: p fruits[0]. To print to the screen "pear" we type: p fruit[2]
