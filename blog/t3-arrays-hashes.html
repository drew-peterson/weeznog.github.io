Hashes vs Arrays

Arrays

Creating New Array

Arrays are ordered collections of objects. A common use for arrays is to group similar objects in a list and give them an ordered number starting with 0. You can create a new array by typing in the following code into your ruby text editor:
fruits = ["apple", "orange", "pear"]

Adding objects to array

In our example above we created a new array called fruits. We can add more fruits to our array with the following code, here are several different ways:
fruits.push("lemon", "lime")
    => ["apple", "orange", "pear", "lemon", "lime"]

fruits.unshift("lemon", "lime")
    => ["lemon","lime","apple", "orange", "pear"]

Selecting objects in array

Every value in an array is assigned an index number. in our array fruits = ["apple", "orange", "pear"] the value "apple" has a index value of 0. We can retrieve and print to the screen "apple" with the following code: p fruits[0]. To print to the screen "pear" we type: p fruit[2]

We can also select existing array indexs and change value held at that index by selecting the index position and setting it equal to a new value. So lets say to want to replace "orange" with "grape" we would type: fruits[1] = "grape"

Hash

Creating New Hash

Hash are similar to arrays in that they too are a ordered collection. However instead of accessing values using index positions, each value is stored in a pair, we call this a key and value pair. If we wanted to create a hash that store the name of a fruit and its color we would do that like this:
fruit = {
    apple: "red",
    orange: "orange",
    pear: "green"
}

Adding Objects to Hashes

Because hashes are created in pairs, there is no real order and thus add a new pair key value pairs to an existing hash is done like this: fruit[:grape] = "purple" and fruit[:lemon] = "yellow".

Selecting and changing values

Selecting and changing Hash values is very similar to adding new key value pairs. In the fruit hash above we want to print to the screen the value "orange", to do this we select the key: puts fruit[:orange] #=> "orange". If we want to see what the key is for a particular value we use the method .key("value"). So to see what key corresponds to the value "green" we do fruit.key("green") #=> :pear

Changing values is just like adding new values we select the key we want to change and add a new value: fruit[:red] = "strawberry"

Different Hash Syntax

For our examples above we used symbols :apple for our keys. Symbols are good for hashes because they are difficult to change and the syntax is easier to write. Below here is another way to create a new hash using strings and hash rockets =>


vegies = {
    "lettuce" => "green",
    "onion" => "white",
    "eggplant" => "purple"
}

To select and change hash value store using the method above you must target the key value in string form: vegies["lettuce"] #=> green
