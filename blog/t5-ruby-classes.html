class Lamp
    def initialize(watts, shade_style)
        @watts = watts
        @shade_style = shade_style
        @light_span = rand(1..5)
        @wear_and_tear = 0
        @powered = false
    end

    def on
        if @powered == true
            puts "Its already On"
        else
            puts "You Turn the Lights On"
        @powered = true
        end
    end

    def off
        if @powered == false
            puts "Its already off"
        else
            puts "you turn the Lights off"
            @powered = false
            broken_light
        end
    end

    def broken_light
        @wear_and_tear += 1
        if @light_span == @wear_and_tear
            puts "your light broke.. Its time to change the bulb..."
            puts "you take off your #{@shade_style} and
             replace it with another #{@watts} watt bulb."
        end
    end
end

.group_by{|element|block} is a method that can be called on collection object. Each element is passed to the block as a parameter. The block goal is to organize elements into a hash based the value returned from the block. A hash has 2 properties, keys and values. The key would be the return value from passing element to the block. The value will be an array of elements that match the key.


note:How the keys are all the values from the previous hash and the values include the keys from the previous hash along with the values. Unfortunatly group_by takes each key, value pair and places them in their own array. We can clean this up with additonal code, but the most important thing to remeber from .group_by is it always returns a hash. And the key will equal the return value of the black. Value will equal an array of the entire element that match true for the return value.

If you want to return 2 arrays, the 1st array containing the elements that return True for the block, and everything else in the 2nd array, check out partition.
