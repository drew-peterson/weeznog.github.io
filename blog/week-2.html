Week 2

Postions: Relative, Absolute, Fixed

Relative

  • The element will be intially be placed exactly where it would have been placed within the normal work flow.
    • You can then move the element using specific key words (top, bottom, left, right) with a combination of lengths of measurement (px, em, %).
  • For example: you want to move a block with the class "red" 4px down and 10px right and give it a background color of blue
    • .red {position: Relative; top: 4px; left: 10px; background-color: blue;}
    • English translation:position the element with the class 'red' exactly where it would have been placed. The position the element 4px from the top of the element to its orginal location. Then position the element 10px measuring for the left side to its orignal location.
  • Moving outside the containing div will not affect placement.

Absolute

  • when the positioning property is given a value of absolute. the element is taken out of normal flow and no longer affects the position of others elements. they act like they are not there.
    • Example: you want to place a div on the very right hand side of the screen and take up the entire height from top to bottom.
      • div {position: absolute; right: 0; top: 0; height: 100%;}
      • English translation:for the element div, take it out of the work flow, make the box start on the very upper right of the screen with no gap. Then take up the entire height.

Fixed

  • Type of absolute positiong that positions the element in relation to the browser window. Therefore, when a user scrolls down the page, it stays in the exact same place.
    • Place footer at the very bottom of screen and give it a height of 15px; and a width of 100%
      • footer {position: fixed; bottom: 0; width: 100%; height: 15px;}
      • English translation:for the element footer, placed the element 0px from the very bottm of the screen and make sure it is always visible in the exact same place not matter if i scroll up or down. Then make it take up the entire width of the screen. Finnaly make it 15px tall.
      • **The height will be important later if you have alot of content above it.
