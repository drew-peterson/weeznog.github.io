for the Pokemon table select all items in column "Name" and "Type"
SELECT Name,Type
FROM Pokemon;
use select keyword and the name of both columns you want to display. Make sure you use the table name found using .schema
display the id and name for electric type pokemon. order them by pokemon id smallest to largest
SELECT PokemonId,Name,Type
FROM Pokemon
WHERE Type="Electric"
ORDER BY PokemonId;
order by defaults to ascending order. Make sure you have a ';' on the very last line.
Display Pokemon name where the height is between 1 and 2 feet tall.
SELECT Name
FROM Pokemon
WHERE Height between 1 and 2;
- when you use between for dates, use hashes.
- for strings you can you "word%" to find anything that starts with word.
display all pokemon where the name ends in 'r'
SELECT Name
FROM Pokemon
WHERE Name like "%r";
- "%key%" : would look before and after the keyword or letter.
- "key%" : would look for cells that start with keyword or letter.
display entire table
SELECT *
FROM Pokemon;
you can add additional filters such as order by and where...
remove all pokemon where is type is Grass
DELETE FROM Pokemon
WHERE Type="Grass";
If you want to delete another something you can include: AND ColumnName = value;
